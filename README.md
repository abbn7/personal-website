# Personal Website

Modern personal portfolio website for Abdelhamed Nada featuring a clean, tech-inspired design with gradient cards and responsive layout.

## Features
- Modern UI with subtle gradients
- Responsive design for all devices
- Tech/AI-inspired color palette
- Interactive sidebar navigation
- Feature cards with colored icons

## Design Specifications
- **Color Palette**: Professional tech colors with subtle gradients
- **Typography**: Inter font family
- **Layout**: Mobile-first responsive design
- **Components**: Rounded cards, pill-shaped buttons, smooth shadows

## Technologies
- HTML5
- CSS3 with custom properties
- Vanilla JavaScript
- Google Fonts (Inter)

Live at: https://abbn7.github.io/personal-website/
